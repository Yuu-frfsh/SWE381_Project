This project aims to implement a web application to facilitate organizing soccer matches.

Two actors are involved:

1-The stadium owner. He should be able to

- sign up/sign in and sign out

- add new stadium: its description, location and photos of the stadium and the facilities

- add reservation slots (when the stadium is available for reservation during the 7 upcoming days)

- display the current reservation status (red color for the reserved slots and green for the available slots)

- send and receive messages to/from match organizer

- display statistics about reservations

All the previous features require signing in.

2- The user. He should be able to

- sign up/sign in and sign out

- search stadium by: location, time slot availability

- check the stadium availability schedule

- reserve a stadium for a selected time slot (requires signing in)

- cancel reservation (requires signing in)
